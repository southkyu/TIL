# JS Core : Scope

> Scope란 무엇일까?

다음 코드를 살펴보자!
```js
const name = "Lydia"
const age = 21
const city  = 'San Francisco'

function getPersonInfo() {
	const name = "Sarah"
	const age = 22

	return `${name} is ${age} and lives in ${city}`
}

console.log(getPersonInfo())
```

`이름` `나이` `도시` 값 변수를 포함한 문자열을 반환하는 `getPersonInfo` 함수를 호출한다.
